## Spring Boot WebUI

### Technologies
Spring Boot, MVC, InMemoryHashMap


### Steps
##### Build and Run Application
via spring-boot-maven-plugin:  
*mvn spring-boot:run*

##### Build jar, Run jar
*mvn package*
*java -jar target/jar-name.nar*

##### Access Application
*http://localhost:8080*
